# ⚡ Quick Start

## 📥 Clone Repository

```bash
git clone https://github.com/OnlineUnknowns/AI-Powered-Clinical-Operating-System.git
```

---

## 📦 Install Dependencies

```bash
npm install
```

---

## ⚙️ Configure Environment

```env
DATABASE_URL=
NEXTAUTH_SECRET=
NEXTAUTH_URL=
OPENAI_API_KEY=
```

---

## 🚀 Run Development Server

```bash
npm run dev
```
